# Introduzione #

Ziao!

Benvenuti in ROS_PO

ROS_PO (ROS Paolo Ornitorinco) è il proetto ROS del corso di "Fondamenti di Robotica"


# Installazione #

Clone the repository with

```
git clone https://github.com/DanieleDifra/ROS_PO
```

Remember that this repo is still private.

Then run

```
cd ROS_PO
catkin_make
```
